## Description
This application aims to help new frontend developers.

## Installation

```bash
$ npm install
```

## Running the app

```
Edit .env JWT_SECRET and run this command:

$ npm run start:dev
```

## Opening database
```bash
$ npx prisma studio
```

## Opening Swagger
```
http://localhost:4000/api-docs/
```

## How it works
```
Create an user and you can get, update or delete data, and create posts.
```
